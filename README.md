# Advent of Code 2024
[AoC](https://adventofcode.com/2024) solved in Python :snake:

## Setup
Add your aoc cookie to `cookie.txt` in the root of the project.  

## Execution
### Using make
```sh
make run day=0
```

```sh
make test day=0
```

### Using python
```sh
python main/day00.py
```

```sh
python -m tests.day00 
```

## Profiling
### Using make
```sh
make profile day=0
```

### Using python
```sh
python -m cProfile -s cumulative main/day00.py
```
